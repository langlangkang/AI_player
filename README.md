# AI_player
通过手势识别控制播放器

#首先打开扫描歌代码，按自己需求扫描
#会自动把它们保存在当前目录下music文件夹下

手势对应
1. ok 播放
2. 五指 音量放大
3. 握拳 音量放小
4. 中指 暂停
5. 手指6 继续播放
6. 手指向上 上一首
7. 手指向下 下一首
8. 双手抱拳 退出

打开main主程序即可运行

可能出现的问题：
出现ModuleNotFoundError: 
1.pip install opencv-python 安装 'cv2'
2.pip  install  baidu-api 安装   'api'  
 
百度手势识别的帮助文档链接：http://ai.baidu.com/ai-doc/BODY/Yk3cpymjy
方便规范手势动作和对程序进行修改
